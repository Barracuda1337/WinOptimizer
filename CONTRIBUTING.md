# Contributing to WinOptimizer

First off, thanks for taking the time to contribute!

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tracker.
- Describe the expected behavior and the actual behavior.
- Include your Windows version and any relevant hardware specs.

### Suggesting Enhancements
- Open an Issue with the tag `enhancement`.
- Explain why the feature would be useful.

### Pull Requests
1. Fork the repo.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Coding Style
- Use clear comments.
- Maintain ASCII-only characters in the script for PowerShell compatibility.
- Ensure the script remains idempotent (safe to run multiple times).
