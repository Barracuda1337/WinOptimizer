# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-07-03
### Added
- 17 different optimization modules.
- Dynamic HTML report generation.
- ASCII-safe UI for maximum compatibility.
- `config.json` for easy customization.
- Bloatware removal engine.
- Game Mode and GPU Scheduling optimization.
- Support for Firefox and Edge cache cleaning.

### Changed
- Complete rewrite of the optimization engine.
- Improved error handling and logging.

## [1.0.0] - 2026-07-03
- Initial internal release.
- Basic mouse lag and temp cleanup modules.
