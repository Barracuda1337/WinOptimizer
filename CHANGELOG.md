# Changelog

## [2.3.0] - 2026-07-03
### Added
- **Categorized Software Store:** A massive update adding 30+ applications selectable by categories (Browsers, Dev Tools, Media, etc.).
- **Cursor AI Support:** Added Cursor (Anysphere.Cursor) to the developer tools list.
- **Improved UI:** New hierarchical menu system for easier navigation.
- **Single-App Selection:** Select specific programs (e.g., `1,3`) or install all in a category (`A`).

## [2.2.1] - 2026-07-03
- Fixed empty software list by internalizing the default app database into the script.

## [2.1.0] - 2026-07-03
- Added Winget auto-update for all apps.
- Added Safe Telemetry and Privacy protection.
- Added Power Throttling prevention.

## [2.0.0] - 2026-07-03
- Major rewrite with 17 standalone modules and HTML reporting.
